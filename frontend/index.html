How CareDesk Agent works
CareDesk v2 is a true AI agent — not just a chatbot. Claude reasons over your request, picks the right tool, executes it, and loops until it has a complete answer.
🔁 Agent loop Architecture
Every message runs through a tool-use loop: Claude decides which tool to call (clinic lookup, calendar, escalation, summary), executes it via the FastAPI backend, incorporates the result, and reasons again — until it reaches a final answer.
📅 Real scheduling Cal.com
When Cal.com is configured, the agent checks live availability and creates real bookings. Without it, a mock mode returns realistic demo slots — so the flow works end-to-end in development.
⚠️ Smart escalation Safety
Claude decides when to escalate — not keyword matching. It calls the escalate_to_staff tool with urgency levels (low → emergency), logs it in the sidebar, and can send email alerts via Resend for high-priority cases.
📋 AI-written summary Workflow
The visit summary is generated by Claude, not a template. It produces structured JSON (chief complaint, outcome, risk level, recommended follow-up) based on the full session context.
